# analytics

A reusable analytics service.

## Supported providers

At the moment only the following providers are supported:

- Firebase Analytics

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  analytics: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  analytics:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: analytics
      ref: main
```
