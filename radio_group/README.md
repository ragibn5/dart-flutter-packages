# radio_group

A fully customizable radio group.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  radio_group: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  radio_group:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: radio_group
      ref: radio_group-1.0.0
```

For more information, see the [package on pub.dev](https://pub.dev/packages/radio_group) or the [GitHub repository](https://github.com/Ragibn5/dart-flutter-packages/tree/main/radio_group).
