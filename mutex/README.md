# mutex

A lightweight mutex for simple locking.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  mutex: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  mutex:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: mutex
      ref: main
```
