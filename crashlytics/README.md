# crashlytics

A reusable crashlytics service.

## Supported providers

At the moment only the following providers are supported:

- Firebase Crashlytics

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  crashlytics: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  crashlytics:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: crashlytics
      ref: main
```
