# alerter

A top-level alert dialog presenter.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  alerter: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  alerter:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: alerter
      ref: 1.0.0
```
