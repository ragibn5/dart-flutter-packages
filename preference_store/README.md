# preference_store

Key-value data store.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  preference_store: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  preference_store:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: preference_store
      ref: main
```
