# common_widgets

A collection of reusable Flutter widgets.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  common_widgets: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  common_widgets:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: common_widgets
      ref: common_widgets-1.0.0
```

For more information, see the [package on pub.dev](https://pub.dev/packages/common_widgets) or the [GitHub repository](https://github.com/Ragibn5/dart-flutter-packages/tree/main/common_widgets).
