# core_models

Shared core models.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  core_models: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  core_models:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: core_models
      ref: core_models-1.0.0
```
