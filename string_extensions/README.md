# string_extensions

Useful extensions over dart's built-in `String` data type.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  string_extensions: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  string_extensions:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: string_extensions
      ref: 1.0.0
```
