# sqlite_db

SQLite database.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  sqlite_db: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  sqlite_db:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: sqlite_db
      ref: 1.0.0
```
