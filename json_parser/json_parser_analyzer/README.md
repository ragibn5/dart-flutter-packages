# json_parser_analyzer

Analysis server plugin to enforce json_parser compatible structures.

## Installation

#### From pub.dev (Not yet available, use git based dependency management for now)

Add this to your `pubspec.yaml`

```yaml
dependencies:
  json_parser_analyzer: ^0.0.1
```

#### Or, From Git repo (Internal members only)

```yaml
dependencies:
  json_parser_analyzer:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: json_parser_analyzer
      ref: main
```
