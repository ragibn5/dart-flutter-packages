# snacker

A top-level snack bar presenter.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  snacker: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  snacker:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: snacker
      ref: snacker-1.0.0
```
