# functions

A collection of reusable functions missing in core dart sdk.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  functions: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  functions:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: functions
      ref: 1.0.0
```
