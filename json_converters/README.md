# json_converters

A collection of reusable JSON converter implementations for json_serializable based modules.

## Installation

#### From pub.dev (Not yet available, use git based dependency management for now)

Add this to your `pubspec.yaml`

```yaml
dependencies:
  json_converters: ^0.0.1
```

#### Or, From Git repo (Internal members only)

```yaml
dependencies:
  json_converters:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: json_converters
      ref: main
```
