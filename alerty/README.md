# alerty

A top-level alert dialog presenter.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  alerty: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  alerty:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: alerty
      ref: alerty-1.0.0
```
