# file_store

File-based data storage.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  file_store: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  file_store:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: file_store
      ref: 1.0.0
```
