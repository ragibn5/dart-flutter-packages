# initializable

Collection of reusable initialization contracts and services.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  initializable: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  initializable:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: initializable
      ref: initializable-1.0.0
```
