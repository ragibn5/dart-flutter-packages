# nav_router

Navigation router.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  nav_router: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  nav_router:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: nav_router
      ref: 1.0.0
```
