# disposable

Collection of reusable disposition contracts and services.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  disposable: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  disposable:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: disposable
      ref: disposable-1.0.0
```
