# datetime_utils

Collection of extensions and utils specific to Date & Time manipulation.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  datetime_utils: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  datetime_utils:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: datetime_utils
      ref: datetime_utils-1.0.0
```
