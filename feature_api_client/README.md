# feature_api_client

Reusable API client with typed request/response contracts and error handling.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  functionals: ^1.0.0
  feature_api_client: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  functionals:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: functionals
      ref: functionals-1.0.0
  feature_api_client:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: feature_api_client
      ref: feature_api_client-1.0.0
```

> **Note:** The `functionals` package is required.
