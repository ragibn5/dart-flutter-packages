# feature_api_client

Reusable API client with typed request/response contracts and error handling.

## Installation

#### From pub.dev (Not yet available, use git based dependency management for now)

Add this to your `pubspec.yaml`

```yaml
dependencies:
  feature_api_client: ^0.0.1
```

#### Or, From Git repo (Internal members only)

```yaml
dependencies:
  feature_api_client:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: feature_api_client
      ref: main
```
