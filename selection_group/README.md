# selection_group

A fully customizable selection group.

## Installation

#### From pub.dev

Add this to your `pubspec.yaml`

```yaml
dependencies:
  selection_group: ^1.0.0
```

#### Or, From Git repo

```yaml
dependencies:
  selection_group:
    git:
      url: https://github.com/Ragibn5/dart-flutter-packages.git
      path: selection_group
      ref: selection_group-1.0.0
```

For more information, see the [package on pub.dev](https://pub.dev/packages/selection_group) or the [GitHub repository](https://github.com/Ragibn5/dart-flutter-packages/tree/main/selection_group).
